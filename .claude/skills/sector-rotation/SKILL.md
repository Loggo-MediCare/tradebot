# /sector-rotation — 產業輪動分析

## 用法
```
/sector-rotation
/sector-rotation tw
/sector-rotation us
```

## 說明
分析目前哪些產業資金流入最強，判斷市場輪動方向，找出下一個主流強勢族群。

## 執行步驟

1. **台股產業分類**（從現有信號腳本分組）：
   ```
   半導體：  2330 台積電, 2303 聯電, 2454 聯發科, 3661 世芯, 2344 華邦電...
   電子零件：2317 鴻海, 2382 廣達, 3231 緯創, 4938 和碩...
   PCB：     3037 欣興, 2383 台光電, 8046 南電, 6274 台燿...
   被動元件：2327 國巨, 2345 智邦...
   金融：    2881 富邦金, 2884 玉山金...
   航運：    2603 長榮...
   ```

2. **計算各產業的平均成交金額與信號強度**：
   - 從最近信號輸出中抓各股成交金額
   - 計算產業平均 AI 信號強度（BUY信號比例）
   - 計算產業平均 MA50 斜率（趨勢強度）

3. **輸出產業熱力圖**：
   ```
   🔥 產業輪動分析 — 2026-05-27
   ════════════════════════════════════════
   排名  產業       平均成交金額   買入比例  MA50趨勢
     1   半導體     NT$8.2B/股    75%      🟢強勢上升
     2   PCB板      NT$3.1B/股    60%      🟢上升
     3   電子組裝   NT$5.8B/股    40%      🟡盤整
     4   金融股     NT$1.2B/股    25%      🟡盤整
   ════════════════════════════════════════
   💡 結論：資金持續流入半導體，PCB 開始跟進
   ⚠️  警示：電子組裝類資金動能減弱，注意輪動
   ```

4. 對照上週數據（如有）判斷輪動方向：哪個產業資金在流入、哪個在流出
