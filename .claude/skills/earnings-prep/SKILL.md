# /earnings-prep — 財報前分析備忘錄

## 用法
```
/earnings-prep <股票代碼>
```

## 範例
```
/earnings-prep 2330
/earnings-prep AMD
/earnings-prep 2454
```

## 說明
在財報公布前，生成完整的分析備忘錄，包含市場預期、關鍵觀察指標、可能的價格反應情境。

## 執行步驟

1. **抓取財報資訊**（yfinance）：
   ```python
   import yfinance as yf
   t = yf.Ticker('<代碼>.TW')
   calendar = t.calendar        # 財報日期
   earnings_history = t.earnings_history  # 歷史財報 EPS
   analyst_estimates = t.analyst_price_targets
   ```

2. **生成財報備忘錄**，格式如下：

   ```
   ╔══════════════════════════════════════════════════════╗
   ║  📋 財報前分析備忘錄 — 2330 台積電                    ║
   ║  財報日期: 2026-Q2  |  當前價格: NT$1,050            ║
   ╚══════════════════════════════════════════════════════╝

   📊 市場預期
   ─────────────────────────────
   EPS 預期:      NT$xx.xx（分析師一致預期）
   營收預期:      NT$xxx 億
   毛利率預期:    xx%
   vs 上季實際:   EPS NT$xx.xx（+/-xx%）

   🔍 本次財報關鍵觀察指標
   ─────────────────────────────
   1. AI 伺服器需求：CoWoS 先進封裝產能利用率
   2. 3nm/2nm 良率提升進度
   3. 美國廠進度與成本
   4. 下季度 Guidance 是否超預期

   📈 歷史財報反應（過去 4 季）
   ─────────────────────────────
   Q1-2026:  EPS NT$xx  →  隔日 +/-xx%
   Q4-2025:  EPS NT$xx  →  隔日 +/-xx%
   Q3-2025:  EPS NT$xx  →  隔日 +/-xx%
   Q2-2025:  EPS NT$xx  →  隔日 +/-xx%

   🎯 情境分析
   ─────────────────────────────
   🟢 超預期（EPS > 預期 +5%）：預估股價 +5~8%，目標 NT$xxx
   🟡 符合預期（±5%）：       股價 ±2%，繼續觀察趨勢
   🔴 不如預期（EPS < -5%）：  預估股價 -5~10%，支撐 NT$xxx

   ⚠️  風險提示
   ─────────────────────────────
   • 地緣政治風險（美中科技戰）
   • 匯率影響（台幣升值）
   • 客戶庫存去化進度
   ```

3. 結合 `/tw-signal` 或 `/us-signal` 的當前 AI 信號，判斷「財報前是否適合建倉」
