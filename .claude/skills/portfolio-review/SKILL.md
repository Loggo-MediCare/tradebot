# /portfolio-review — 持倉健康度評估

## 用法
```
/portfolio-review
```

## 說明
評估目前持有部位的整體健康狀況，檢查風險集中度、信號變化，提供加減碼建議。

## 執行步驟

1. **讀取持倉資料**：
   - 從記憶檔案讀取：`C:\Users\Silvi\.claude\projects\...\memory\`
   - 或請使用者輸入目前持倉（股票代碼 + 持有成本 + 張數）

2. **對每個持倉股票執行信號評估**：
   - 找對應的 `get_trading_signal_xxx.py`
   - 執行並取得最新信號
   - 計算：未實現損益 = (現價 - 成本) × 張數 × 1000

3. **風險分析**：
   ```
   📊 持倉健康度報告
   ═══════════════════════════════
   總市值:     NT$xxx萬
   未實現損益: +/- NT$xxx萬 (+/-xx%)
   
   ⚠️  風險警示：
   • 半導體類股集中度 xx%（建議 <40%）
   • xxx 信號已轉為賣出，建議減碼
   • xxx RSI 超買 >75，注意回調風險
   
   ✅  健康持倉：
   • xxx  🟢買入信號持續，可續抱
   • xxx  MA50上升趨勢穩健
   
   🎯  建議操作：
   1. 減碼 xxx（賣出信號，建議減 20-30%）
   2. 加碼 xxx（買入信號+放量突破）
   3. 觀望 xxx（等待方向確認）
   ```

4. **產業分散度分析**：依照台積電/鴻海/面板/金融/電子等分類，計算各類股比重
