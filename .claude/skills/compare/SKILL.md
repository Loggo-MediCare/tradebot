# /compare — 多股比較分析（Comp Analysis）

## 用法
```
/compare <代碼1> <代碼2> <代碼3>...
```

## 範例
```
/compare AMD MU INTC NVDA
/compare 2330 2303 2454
/compare 2382 2317 3231
```

## 說明
對多支股票進行橫向比較分析，類似投資銀行的 Comparable Company Analysis（可比公司分析）。

## 執行步驟

1. **逐一抓取每支股票資料**（yfinance）：
   - 基本資訊：市值、P/E、P/B、EV/EBITDA
   - 技術面：當前價格、RSI、MACD、MA50斜率
   - AI信號：從各 `get_trading_signal_xxx.py` 的最新輸出判斷

2. **輸出比較表格**：
   ```
   ╔══════════════════════════════════════════════════════════════╗
   ║  Comparable Analysis — AMD / MU / INTC / NVDA              ║
   ╠══════════╦═══════╦═══════╦═══════╦═══════╦════════════════╣
   ║  指標    ║  AMD  ║  MU   ║  INTC ║  NVDA ║  備註          ║
   ╠══════════╬═══════╬═══════╬═══════╬═══════╬════════════════╣
   ║  價格    ║$503   ║$895   ║$123   ║$xxx   ║                ║
   ║  市值    ║$815B  ║$xxx   ║$xxx   ║$xxx   ║                ║
   ║  P/E     ║  35x  ║  22x  ║  18x  ║  xxx  ║ MU 最低        ║
   ║  ROE     ║  45%  ║  28%  ║  5%   ║  xxx  ║ AMD 最強       ║
   ║  毛利率  ║  53%  ║  38%  ║  42%  ║  xxx  ║                ║
   ║  AI信號  ║  🟢   ║  🟢   ║  🟡   ║  xxx  ║                ║
   ║  分析師  ║強烈買 ║強烈買 ║ 持有  ║  xxx  ║                ║
   ╚══════════╩═══════╩═══════╩═══════╩═══════╩════════════════╝
   ```

3. **排名與結論**：
   - 🏆 最佳估值（P/E最低）
   - 🏆 最強獲利（ROE最高）
   - 🏆 最強AI信號
   - 📌 綜合推薦排序

4. 如果是同產業，額外計算各股相對於產業平均的溢/折價幅度
